# Doc4Team

### 目标

Doc4Team是为了解决API文档的快速查询；一般文档管理只对方法名，API名称等做了模糊查询。

当你想要一个功能然而对于方法名、API名称、归属项目、作者等不知情该如何查询？茫茫文档中无法查询那就只能：
- 问队友
- 翻看代码可能使用到的地方
- 造轮子

这便是Doc4Team要解决的API文档查询问题。
